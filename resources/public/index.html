Frajola
Projeto para tirar fotos com camera DSLR e postar no twitter

Este projeto foi concebido para tirar fotos de passaros de forma inteligente e simples.

Projeto


usb

Raspberry & GPIO

Usando o Raspberry PI para executar o código responsável por medir a distância entre o objeto e chamar o programa que tira foto e enviar para o twitter.
Através do GPIO e um sensor ultrasonito, medimos a distância de um objeto e ao chegar em uma determinada distância, é disparado uma foto do objeto.

photo_camera

Tirar Foto

Usando a biblioteca gphoto2 controlamos a câmera DSLR. Que ao ser feita uma requisição na app, dispara a foto e faz o download da câmera.

settings

Enviar para Twitter

Através da biblioteca twitter-api é feito um novo post no twitter.
Para fazer o post no twitter é preciso ter uma app criada no twitter.

Aplicação que dispara foto e envia para twitter

Clojure

Usando a biblioteca org.gphoto/gphoto2-java fazemos a comunicação com o gphoto2 que é responsável por tirar as fotos e fazer o download para o raspberry-pi.
Depois usamos twitter-api para enviar a imagem para o twitter
As configurações das chaves e ookens de acesso é feito através de variáveis de ambientes, nas quais são destritas no README do projeto frajola
Para realiar o disparo da foto é necessário chamar a url http://localhost:3000/take-picture. A abordagem de fazer o disparo via url foi feita para dimunuir o tempo de load da aplicação e o disparo ser feito de forma rápida, além de proporcionar o disparo remoto.

Raspberry-pi medindo distância de objetos

Raspberry-pi & Python

O raspberry-pi além de embarcar o nosso código usamos o recurso do GPIO para acoplar um sensor ultrasonico, que nos permite medir a distância de um objeto com uma precisão muito boa.
Usamos python para acessar as portas do GPIO e calcular a distância de um objeto. Um objeto ao atingir a distância determinada é feito a chamada da url que dispara a fotoe encaminhar para o twitter. Os arquivos python estão na pasta /lib do projeto. no qual deve ser executado o main.py. Caso a distancia queira ser alterada sem precisar acessar o raspberry, a mudança pode ser feita através do protocolo MQTT.
Para alterar é só passar o novo valor em centimentros para o tópico frajola/distance. Por default o valor é 40 centimetros. As configurações do mqtt também é feita através de variáveis de ambiente, descritas no README do projeto
